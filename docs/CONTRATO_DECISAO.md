## Contrato de Decisão – v1

Patterns suportados:
- IMPACTO_OPERACIONAL → 1 foco (max impact)
- RECORRENCIA_CATEGORIA → lista
- ANALISE_EQUIPAMENTO → histórico
- ANALISE_GERAL → resumo

O LLM não pode:
- Criar dados
- Inferir categorias
- Listar gargalos múltiplos

Fonte da verdade: motor analítico.
